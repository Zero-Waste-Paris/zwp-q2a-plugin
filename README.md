# Installation

- créer le dossier `fonts` et y ajouter un fichier `font.ttf` (copie ou lien symbolique);
- ajout les fichiers https://github.com/stil/gd-text/tree/master/src dans le dossier `GDText` (à créer dans le dossier du plugin);
- appliquer le patch `gd-text.patch` au fichier `GDText/Box.php` (`patch Box.php gd-text.patch`).
